# 🚌 Bus Booking System

A web-based bus booking system developed using PHP, MySQL, HTML, CSS, and JavaScript.

---

## 🚀 Features

- User Registration & Login
- Bus Search & Booking System
- Seat Selection
- Admin Panel (Add/Delete Buses)
- Contact & About Page

---

## 🛠️ Technologies Used

- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  

---

## 📂 Project Structure

- index.php → Home page  
- book.php → Booking page  
- seats.php → Seat selection  
- admin_login.php → Admin login  
- dashboard.php → Admin dashboard  
- add_bus.php → Add new bus  
- delete_bus.php → Delete bus  
- db.php → Database connection  

---

## ⚙️ How to Run

1. Install XAMPP/WAMP  
2. Copy project folder to `htdocs`  
3. Start Apache & MySQL  
4. Import database in phpMyAdmin  
5. Run project on `localhost`
